一. 买卖股票

我的代码：

暴力法

  1. ```java
    //暴力法： for二重循环 两指针索引 遍历
    //缺点 ： 时间慢， 有重复计算
    //时间复杂度： O(n^2) 空间复杂度 O(1)–一个常量
    public static int maxProfit(int[] prices) {
        int max=0;
    for(int i=0;i<prices.length;i++){
        for(int j=prices.length-1;j>i;j--){
            if(prices[j]-prices[i]>max) {
                max = prices[j] - prices[i];
            }
        }
    }
    return max;
}
